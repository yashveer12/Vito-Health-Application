# vito (temporary name: xyz)

Simple web app to select an organ and upload a report. Uploads are stored under `uploads/<organ>/`.

Quick start

```bash
cd Projects/vito
npm install
npm start
# open http://localhost:3000
```

Notes
- Supported organs: Heart, Pancreas, Liver, Blood, Lungs, Kidney
- Uploaded files are saved in `uploads/<organ>`
